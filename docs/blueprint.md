# **App Name**: AlertNet

## Core Features:

- User Authentication: Secure user login and registration with admin role. Demo user: demo@gmail.com, password demo@0123. Admin login: Admin@gmail.com, password Admin@0426.
- User Profile Management: Allow users to manage their profile information.
- Alert Creation: Users can create alerts by setting location and uploading images of micro-disasters.
- Disaster Verification: Use Gemini AI to verify if an uploaded image represents a disaster. The AI will act as a tool, reasoning whether the given information points to a micro-disaster event.
- Alert Display: Display verified disasters with name, severity, description, and image.
- Real-time Notifications: Display small popup notifications to all users when a new disaster is added.
- Emergency Contact Management: Provide a list of emergency numbers and allow users to add custom numbers.
- Admin Alert Management: Enable admins to manage alerts and send notifications to all users.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey trust and reliability.
- Background color: Light gray (#F0F2F5), nearly desaturated blue, for a clean, professional look.
- Accent color: Vibrant orange (#FF7733), analogous to red, for high-visibility alerts and calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text.
- Use clear, consistent icons to represent disaster types and alert levels.
- Use a card-based layout to display alerts and analytics in a structured manner.
- Implement smooth transitions and subtle animations for a polished user experience.
